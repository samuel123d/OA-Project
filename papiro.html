Tutorial dos Papiros

Como jogar

Primeira pagína:

  1. A primeira página apesenta quais os números que serão trabalhados.
  2. Enquatos você lê sobre o contexto historico de roma e apredende nome de tropas romanas.
  3. Aprete em continuar para ir para a próxima fase.

Segunda página:

Na segunda página você deve arrastar os números para o local correto de acordo com a carta que leu na primeira página. Você tem 3 tentativas para errar, caso erre 3 vezes, será redirecionado para uma página de game over. Caso acerte todos, um botão de continuar aparecerá para você avançar no jogo.
